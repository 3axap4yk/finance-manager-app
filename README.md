# Управление личными финансами (Finance Manager)

Приложение на Java для ведения личных финансов с поддержкой доходов, расходов, бюджетов и статистики.

Данные каждого пользователя сохраняются в отдельный JSON-файл (`data/<email>.json`)

---
## Возможности

* Авторизация и регистрация пользователей по `email` и паролю
* Хранение доходов и расходов с привязкой к категориям
* Создание и управление категориями доходов и расходов
* Установка и редактирование бюджета для категорий расходов
* Подсчет общей суммы доходов и расходов, статистика по категориям
* Оповещения при превышении бюджета
* Сохранение данных в `JSON` при выходе. Загрузка данных из `JSON` при авторизации

---
## Требования

* Java 17+
* Gson (`com.google.code.gson:gson:2.10.1`)
---

## Команды

### Меню авторизации

```bash
1. Войти
2. Зарегистрироваться
0. Выход
```

### Главное меню

```bash
1. Изменить доходы
2. Изменить расходы
3. Показать статистику
4. Управление категориями
0. Сохранить и выйти
```

### Меню доходов

```bash
1. Добавить категорию дохода
2. Добавить доход
0. Назад
```

### Меню расходов

```bash
1. Добавить категорию расхода
2. Добавить расход
0. Назад
```

### Меню управления категориями

```bash
1. Переименовать/изменить бюджет категории дохода
2. Переименовать/изменить бюджет категории расхода
3. Удалить категорию дохода
4. Удалить категорию расхода
5. Показать категории
0. Назад
```
